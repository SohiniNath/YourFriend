# YourDOST - Therapy Continuity Engine

This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- **Assessment Flow**: AI-driven questionnaire to understand user needs.
- **Matching Engine**: Matches users with counselors based on style and language.
- **Dashboard**: Track sessions and view summaries.
